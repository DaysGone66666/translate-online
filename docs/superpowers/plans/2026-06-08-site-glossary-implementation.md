# 站点术语库实现计划

**目标：** 在现有翻译请求链路前增加站点术语库，支持 HLTV 固定短语和用户自定义术语。

**架构：** 新建独立 `src/glossary.js` 负责术语解析、站点匹配、完整短语查找、句内术语提取和结果校正。内容脚本向服务线程传递当前主机名，服务线程将术语命中接入缓存、免费引擎和 DeepSeek 请求。

**技术栈：** Chrome MV3、原生 JavaScript、Node.js `node:test`
