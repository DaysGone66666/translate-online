# 网页悬浮工具条实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将现有网页右侧翻译按钮升级为可拖动、自动半隐藏的小头像菜单，仅悬停头像时展开翻译与设置按钮。

**架构：** 新建独立的工具条核心模块处理位置限制和拖动判断；内容脚本负责 DOM、样式、指针事件和按钮行为；位置保存在本地扩展存储。

**技术栈：** Chrome MV3、原生 JavaScript、Pointer Events、Node.js `node:test`
