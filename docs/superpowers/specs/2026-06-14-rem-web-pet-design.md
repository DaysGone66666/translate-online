# 雷姆网页宠物设计

## 目标

将现有右侧半隐藏悬浮头像升级为可交互的雷姆网页宠物，同时保留页面翻译、侧边栏、设置入口、拖动定位和站点禁用功能。

宠物采用带背景的竖向状态卡片，不制作透明全身精灵，也不引入持续运行的逐帧动画。

## 素材

源图位于 `images/雷姆图片/`，原图保持不变。构建宠物资源时生成裁剪和压缩后的 WebP 文件：

| 状态 | 源图 | 用途 |
|---|---|---|
| `idle` | `DM_20260614195418_003.png` | 默认待机 |
| `idle-alt` | `DM_20260614195418_005.png` | 长时间待机时的阅读状态 |
| `hover` | `DM_20260614195418_001.png` | 鼠标进入、菜单展开 |
| `loading` | `DM_20260614195418_002.png` | 页面翻译进行中 |
| `success` | `DM_20260614195418_001.png` | 翻译成功 |
| `error` | `DM_20260614195418_004.png` | 翻译失败 |

每张派生图统一为相同纵横比和人物视觉位置，最长边不超过 384 像素。资源应控制在适合扩展随页面加载的大小，避免直接加载五张约 2.3MB 的原图。

## 外观

- 收起状态：右侧只露出约一半宠物卡片，尺寸约 `64×84px`。
- 展开状态：完整显示宠物，并在上方显示翻译按钮、下方显示设置按钮。
- 卡片使用圆角、轻微描边和阴影；图片使用 `object-fit: cover`。
- 待机时使用缓慢呼吸和上下浮动效果。
- 状态切换使用淡入淡出，不做连续精灵帧动画。
- `prefers-reduced-motion: reduce` 时关闭呼吸、浮动和状态过渡。

## 交互

- 鼠标进入宠物：展开工具栏，显示 `hover` 状态。
- 鼠标离开：延迟收起并恢复当前业务状态。
- 单击宠物：打开翻译记录侧边栏。
- 单击翻译按钮：开始页面翻译；翻译中再次点击则取消。
- 单击设置按钮：打开扩展设置页。
- 从宠物卡片拖动：改变垂直位置并保存，拖动结束后不触发侧边栏。
- 键盘聚焦：与鼠标进入一致地展开工具栏。

## 状态

宠物视觉状态由业务状态和临时交互状态共同决定：

1. `loading` 优先级最高，页面翻译期间一直显示。
2. `error` 在翻译失败后显示，并出现错误气泡。
3. `success` 在翻译完成后显示，并出现成功气泡。
4. 非业务状态下，展开时显示 `hover`。
5. 其余时间显示 `idle`，长时间无操作可切换一次 `idle-alt`。

成功和失败状态显示数秒后恢复 `idle`。用户开始拖动时暂停自动状态恢复，拖动结束后继续。

## 气泡

宠物左侧显示短气泡，不遮挡网页主要内容：

- 待机：默认不显示。
- 悬停：显示“需要翻译吗？”。
- 翻译中：显示“正在翻译...”。
- 成功：显示“翻译完成”。
- 失败：显示服务返回的简短错误信息。

气泡使用 `role="status"` 和 `aria-live="polite"`。错误文本限制长度，完整错误仍由现有页面提示机制负责。

## 架构

- `src/floating-toolbar-core.js`
  - 增加纯函数，负责根据业务状态、展开状态和临时状态计算宠物视觉状态。
  - 增加状态对应的图片和气泡描述映射。
- `src/content-script.js`
  - 渲染双图片层以实现淡入淡出。
  - 将现有 `setBallState` 与宠物状态同步。
  - 管理悬停、定时恢复、拖动和气泡生命周期。
- `images/pet/`
  - 保存从五张源图生成的压缩派生资源。
- `tests/floating-toolbar-core.test.js`
  - 测试状态优先级、状态恢复和展示描述。
- `tests/content-style.test.js`
  - 测试宠物结构、尺寸、半隐藏、动画降级和现有入口不被移除。

## 性能

- 不使用持续 JavaScript 帧循环。
- 动画只使用 CSS transform 和 opacity。
- 非当前状态图片不重复解码；浏览器缓存本地扩展资源。
- 页面隐藏时 CSS 动画暂停。
- 原始五张 PNG 不通过 `web_accessible_resources` 暴露，仅派生宠物资源由内容脚本访问。

## 兼容与回退

- 派生图片加载失败时回退到现有 `images/page-ball-avatar.png`。
- 不改变现有存储键 `floating_toolbar_top`。
- 不增加新的浏览器权限。
- 不改变页面翻译 API、供应商配置或侧边栏数据结构。

## 验收标准

1. 宠物在普通网页中默认半隐藏，悬停后完整展开。
2. 翻译、侧边栏和设置三个入口仍可使用。
3. 宠物可垂直拖动，刷新页面后位置保持。
4. 翻译中、成功和失败都有明显不同的图片与气泡反馈。
5. 状态切换没有明显闪白或布局跳动。
6. 480px 宽视口中不产生横向滚动。
7. 开启减少动态效果后无呼吸和浮动动画。
8. `npm test`、`npm run check` 和 `git diff --check` 通过。
