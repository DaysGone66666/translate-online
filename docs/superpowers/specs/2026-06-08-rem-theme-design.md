# 雷姆主题视觉设计

## 目标

在不改变翻译功能和交互结构的前提下，将扩展统一为深蓝、冰蓝与半透明玻璃质感的雷姆主题。

## 资源分配

- `images/options-background.jpg`：设置页背景，使用女仆装雷姆竖版插画。
- `images/sidebar-background.jpg`：侧边栏背景，使用圣诞雷姆竖版插画。
- `images/icon-source.jpg`：扩展图标源图，从中央极简雷姆头像区域裁切。
- `icons/icon16.png`、`icons/icon48.png`、`icons/icon128.png`：由图标源图生成。

## 视觉方案

设置页使用固定全屏背景，右侧保持角色主体可见；内容卡片改为深蓝半透明玻璃面板，使用冰蓝高亮和清晰的浅色文字。

侧边栏使用圣诞插画作为全屏背景，叠加深蓝渐变遮罩。历史记录卡片使用高透明度深色玻璃，输入区和顶部栏保持稳定可读。

划词弹窗继续使用项目现有背景资源，但统一为同一套深蓝、冰蓝、柔光玻璃视觉，避免网页上的弹窗与扩展界面风格割裂。

网页右侧的一键翻译按钮使用极简雷姆圆形头像。默认状态保持窄小，悬停或键盘聚焦时向左展开并显示当前操作；翻译中使用冰蓝旋转光环，翻译完成后显示原文或译文切换状态。

页面内注入的译文保持弱视觉权重：不使用卡片背景、圆角、阴影或固定宽度，仅在原文下方使用继承色文字与细冰蓝左线。译文作为辅助信息融入原页面，而不是形成独立面板。

## 可用性

- 所有正文和控件文字保持足够对比度。
- 焦点状态使用明亮冰蓝描边。
- 保留响应式布局与减少动态效果设置。
- 背景图片仅作装饰，不影响控件语义与键盘操作。
